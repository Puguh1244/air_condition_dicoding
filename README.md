# Air Quality Dashboard (Streamlit)

Dashboard interaktif untuk analisis kualitas udara berdasarkan dataset
Air Quality (PRSA Dataset).

------------------------------------------------------------------------

## A Setup Environment

### 1. Masuk ke folder project kamu
    Pastikan di folder tersebut sudah tersimpan
    1. requirements.txt
    2. dataset
    3. file.py

### 2. (Optional) Buat virtual environment

    python -m venv .venv

### 3. Aktifkan environment

    .venv\Scripts\activate

### 4. buka cmd dari file tersebut

    Lalu ketik pip install -r requirements.txt

------------------------------------------------------------------------

## B Menjalankan Aplikasi
    dalam CMD/Bash setelah install requirements ketik
    "python -m streamlit run NAMA_FILE.py"
